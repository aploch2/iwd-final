## Installation

#### Install via Docker

> Requires: Docker, Docker-Compose, and npm

1. Clone the respository

```
git clone https://www.github.com/aploch2/iwd-final.git
```

2. Navigate to the folder

```
cd iwd-final
```

3. Run the application

```
npm run production
// or
docker-compose up --build
```
